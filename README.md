# PYTHON MINI PROJECTS
A collection of 20 Python mini projects built to strengthen software engineering fundamentals, problem-solving skills, and practical Python development experience.

### `core/mode_manager.py`

**Purpose:**
Manages application navigation and mode switching through a centralized menu system.

**Benefits:**

* Keeps application flow organized
* Makes adding new modes easy
* Promotes scalable and maintainable architecture
* Separates navigation logic from feature logic

---

### `core/tools.py`

**Purpose:**
Provides reusable utility functions for input validation, user prompts, data parsing, and console management.

**Benefits:**

* Reduces duplicated code
* Improves input reliability and user experience
* Centralizes validation logic
* Keeps feature modules clean and focused

---
## MODES/PROJECTS

### `modes/madlibs.py`

**Purpose:**
A console-based Mad Libs game that generates a humorous story using user-provided words and phrases.

**Benefits:**

* Practices user input handling and validation
* Demonstrates object-oriented design through mode inheritance
* Shows string formatting and dynamic content generation
* Introduces game flow and user interaction in a CLI application

### `modes/simple_calculator.py`

**Purpose:**
A command-line calculator that evaluates mathematical expressions entered by the user.

**Benefits:**

* Practices arithmetic operations and expression evaluation
* Demonstrates exception handling for invalid inputs
* Reinforces input validation and user interaction
* Introduces interactive application loops and menu navigation

### `modes/simple_converter.py`

**Purpose:**
A unit conversion tool that converts values between length, weight, and temperature measurements.

**Benefits:**

* Practices data structures using dictionaries and enums
* Demonstrates unit conversion algorithms and mathematical calculations
* Reinforces input validation and user-driven workflows
* Introduces category-based program design and reusable conversion logic

### `modes/compound_interest_calculator.py`

**Purpose:**
A financial calculator that computes the future value of an investment using the compound interest formula.

**Benefits:**

* Practices mathematical formulas and financial calculations
* Reinforces numeric input validation and error prevention
* Demonstrates data processing and result presentation
* Introduces real-world problem-solving through programming

### `modes/countdown_timer.py`

**Purpose:**
A command-line countdown timer that tracks time in hours, minutes, and seconds before displaying a custom completion message.

**Benefits:**

* Practices working with time-based operations
* Demonstrates loops and countdown logic
* Reinforces user input validation and formatting
* Introduces real-time program execution using delays

### `modes/shopping_cart.py`

**Purpose:**
A command-line shopping cart system that allows users to add, edit, remove, and checkout items while calculating the total cost.

**Benefits:**

* Practices object-oriented programming through custom classes
* Demonstrates CRUD (Create, Read, Update, Delete) operations
* Reinforces list management and data manipulation
* Introduces state management and menu-driven workflows

### `modes/quiz_game.py`

**Purpose:**
A multiple-choice quiz game that tests Python programming knowledge and tracks the user's score across questions.

**Benefits:**

* Practices list/tuple data structures and indexing
* Demonstrates state tracking across multiple questions
* Reinforces input validation and control flow
* Introduces scoring systems and result evaluation logic


### `modes/concession_stand.py`

**Purpose:**
A food ordering system that simulates a concession stand where users can select items, manage a cart, and proceed to checkout.

**Benefits:**

* Reuses OOP concepts across modules (`ShoppingItem`)
* Practices dictionary-based menus and dynamic UI generation
* Demonstrates cart management and aggregation logic
* Reinforces real-world workflow simulation (ordering system)
